# Dokuwiki plugin

Este plugin possibilita criar/editar páginas no wiki da Ello tecnologia.

Antes de usar, configure no sistema operacional as variáveis de ambiente:
- `DOKUWIKI_URL`: https://wiki.ellotecnologia.com/lib/exe/jsonrpc.php
- `DOKUWIKI_TOKEN`: seu token de autenticação do wiki

## Como obter o token do wiki

- Acesse o [wiki](https://wiki.ellotecnologia.com)
- Clique no menu **Ferramentas** no canto inferior direito
- Clique no sub-menu **Atualizar Perfil**

O token de autenticação será exibido.
